# Табасаранско-русский переводчик

Веб-приложение для перевода между табасаранским и русским языками.

## Архитектура

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   Vue.js        │────▶│   FastAPI       │────▶│   GPU Server    │
│   Frontend      │     │   Backend       │     │   (LLM)         │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                              │
                              ▼
                        ┌─────────────────┐
                        │   SQLite +      │
                        │   ChromaDB      │
                        └─────────────────┘
```
